## ⚙️ Getting Started  

To run the project locally:  

```bash
# Clone the repository
git clone https://github.com/aayush-506/Angular.git  

# Navigate into the project folder
cd student-dashboard  

# Install dependencies
npm install  

# Run the development server
ng serve
